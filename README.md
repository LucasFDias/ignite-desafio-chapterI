# Conceitos do React


## Objetivo

O Objetivo do desafio era treinar a manipulação do estado no React, onde haviam 3 objetivos:
- Adicionar uma nova tarefa 
- Remover uma tarefa
- Marcar e desmarcar uma tarefa concluída
